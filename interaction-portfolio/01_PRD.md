# Interaction Portfolio Web — PRD (Draft)

## 0. 한 줄 요약
Motto-xx 레퍼런스 감성(미니멀 + 타이포 중심 + 공간감)을 기반으로, **3D/영상 인터랙션을 ‘과시’가 아니라 ‘서사/신뢰’로 쓰는 1페이지 포트폴리오**.

---

## 1. 목표(Goals)
1) 일본 취업/채용(신입~주니어) 관점에서 **“이 사람이 제품을 끝까지 만들 줄 안다”**를 10초 안에 전달
2) 방문자가 스크롤하는 동안, **대표 프로젝트 1~3개**의 임팩트/역할/기술이 자연스럽게 이해되게 구성
3) 3D/영상 인터랙션은 ‘멋’이 아니라 **핵심 메시지(로컬라이징/제품화/속도/품질)**를 강화
4) 연락(LinkedIn/Email)까지의 전환(CTA) 증가

## 2. 비목표(Non-Goals)
- 블로그/아카이브 형태(다중 페이지)로 확장하지 않음 (1 page 유지)
- 복잡한 CMS/관리자 구축은 하지 않음 (초기에는 정적 콘텐츠 중심)
- 과도한 애니메이션(성능 저하, 멀미 유발) 금지

## 3. 타겟 사용자 & 시나리오
### Primary
- (A) Recruiter/HR: “이력서 링크 클릭 → 20~40초 훑고 판단”
- (B) Hiring manager/Tech lead: “프로젝트 진짜 했는지/문제 해결 능력/기술 깊이 확인”

### 핵심 사용 시나리오
1) 링크 오픈 → Hero에서 ‘정체성/포지션/비자(COE) 가능성’까지 맥락 파악
2) 스크롤 → 대표 프로젝트(1~3)에서 역할/성과/기술 스택 확인
3) 신뢰 요소(협업/배송/지표/테스트/운영) 확인
4) CTA 클릭(LinkedIn / Email)

## 4. 핵심 메시지(포지셔닝)
- “제품을 실제로 만들고, 운영하고, 지역/언어/사용자 맥락에 맞게 개선할 줄 아는 개발자”
- 키워드: **Product-minded / Localization(i18n) / Shipped / Performance / Reliable**

## 5. 정보 구조(1 Page)
1) Hero (시그니처 인터랙션)
2) Selected Works (프로젝트 카드 1~3)
3) Impact & Craft (문제→접근→결과, 신뢰 강화)
4) Skills Snapshot (핵심 스킬만)
5) About / Timeline (짧게)
6) CTA / Contact
7) Footer

## 6. 인터랙션 컨셉(레퍼런스: motto-xx 톤)
### 컨셉 A — “Spatial Typography + Orb/Monolith” (선택됨)
- 3D 오브젝트(반투명 구체/모놀리스)가 Hero 배경에서 **미세하게 반응**
- 반응 트리거: 마우스/자이로/스크롤 (강도 낮게)
- 텍스트는 선명, 3D는 ‘공간감’만 제공

### 컨셉 B — “Video as Material”
- 짧은 루프 영상(5~8초) 위에 큰 타이포
- 스크롤 구간에서 영상이 ‘재질’처럼 변형(블러/그레인/매트)

### 반드시 지킬 UX 원칙
- 최초 로드: **1) 텍스트 먼저 2) 미디어는 lazy-load**
- Reduce motion 지원 (prefers-reduced-motion)
- 모바일에서 3D는 품질 자동 하향(또는 비활성) 옵션

## 7. 성공 지표(Success Metrics)
- LCP/INP 등 Core Web Vitals: “합격선” 유지 (특히 모바일)
- CTA 클릭률(LinkedIn/Email): 베이스라인 대비 상승
- Recruiter 시선 기준: 20초 내 “무슨 사람인지/대표 작업이 뭔지” 이해

## 8. 리스크 & 대응
- 3D/영상 용량 과다 → 영상 길이/해상도/코덱 가이드 + 프리뷰 이미지 제공
- 모션 과함 → 모션 강도 옵션 + reduce-motion
- 메시지 과다 → 프로젝트 1~3개만 강하게, 나머지는 링크로

## 9. 콘텐츠 정책(문장 가이드)
- 문장 길이 짧게, 동사 중심
- 숫자/결과가 없으면 “의도 + 범위 + 책임”을 명확히
- COE는 ‘요청’이 아니라 ‘확인’ 톤

## 10. 오픈 이슈(결정 필요)
- 포트폴리오 표시 이름: **EN only** (확정)
- 대표 프로젝트 선정: **SOMETIME + Letter U(프로젝트명) / Lettering(카테고리명)** (확정)
- 연락 채널 우선순위: **LinkedIn 1순위, Email 2순위** (확정)
- 언어 구성: **JP With EN(일어 우선, 레이어로 EN 겹침)** (확정)
